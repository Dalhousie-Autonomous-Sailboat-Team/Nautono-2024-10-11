In this use case, the Watchdog module is configured for\+: \begin{DoxyItemize}
\item System reset after 2048 clocks of the Watchdog generic clock \item Always on mode disabled \item Basic mode, with no window or early warning periods\end{DoxyItemize}
This use case sets up the Watchdog to force a system reset after every 2048 clocks of the Watchdog\textquotesingle{}s Generic Clock channel, unless the user periodically resets the Watchdog counter via a button before the timer expires.

\DoxyCodeLine{1507 \textcolor{comment}{/*}}
\DoxyCodeLine{1508 \textcolor{comment}{ * Queue sets provide a mechanism to allow a task to block (pend) on a read}}
\DoxyCodeLine{1509 \textcolor{comment}{ * operation from multiple queues or semaphores simultaneously.}}
\DoxyCodeLine{1510 \textcolor{comment}{ *}}
\DoxyCodeLine{1511 \textcolor{comment}{ * See FreeRTOS/Source/Demo/Common/Minimal/QueueSet.c for an example using this}}
\DoxyCodeLine{1512 \textcolor{comment}{ * function.}}
\DoxyCodeLine{1513 \textcolor{comment}{ *}}
\DoxyCodeLine{1514 \textcolor{comment}{ * A queue set must be explicitly created using a call to xQueueCreateSet()}}
\DoxyCodeLine{1515 \textcolor{comment}{ * before it can be used.  Once created, standard FreeRTOS queues and semaphores}}
\DoxyCodeLine{1516 \textcolor{comment}{ * can be added to the set using calls to xQueueAddToSet().}}
\DoxyCodeLine{1517 \textcolor{comment}{ * xQueueSelectFromSet() is then used to determine which, if any, of the queues}}
\DoxyCodeLine{1518 \textcolor{comment}{ * or semaphores contained in the set is in a state where a queue read or}}
\DoxyCodeLine{1519 \textcolor{comment}{ * semaphore take operation would be successful.}}
\DoxyCodeLine{1520 \textcolor{comment}{ *}}
\DoxyCodeLine{1521 \textcolor{comment}{ * Note 1:  See the documentation on http://wwwFreeRTOS.org/RTOS-\/queue-\/sets.html}}
\DoxyCodeLine{1522 \textcolor{comment}{ * for reasons why queue sets are very rarely needed in practice as there are}}
\DoxyCodeLine{1523 \textcolor{comment}{ * simpler methods of blocking on multiple objects.}}
\DoxyCodeLine{1524 \textcolor{comment}{ *}}
\DoxyCodeLine{1525 \textcolor{comment}{ * Note 2:  Blocking on a queue set that contains a mutex will not cause the}}
\DoxyCodeLine{1526 \textcolor{comment}{ * mutex holder to inherit the priority of the blocked task.}}
\DoxyCodeLine{1527 \textcolor{comment}{ *}}
\DoxyCodeLine{1528 \textcolor{comment}{ * Note 3:  An additional 4 bytes of RAM is required for each space in a every}}
\DoxyCodeLine{1529 \textcolor{comment}{ * queue added to a queue set.  Therefore counting semaphores that have a high}}
\DoxyCodeLine{1530 \textcolor{comment}{ * maximum count value should not be added to a queue set.}}
